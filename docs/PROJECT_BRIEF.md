# PROJECT_BRIEF.md

## Projektname
DSGVO Website Checker

## Betreiber
That Clicks Marketing · Manuel Schreiner · info@that-clicks.de

## Ziel-URL
https://dsgvo-check.that-clicks.de (Subdomain von that-clicks.de)

---

## Projektziel

Ein kostenloser, öffentlich zugänglicher Website-Check-Service, der kleine Unternehmen
in Deutschland dabei unterstützt, typische DSGVO-, Cookie- und Datenschutzrisiken
auf ihrer Website zu erkennen.

Das Tool fungiert gleichzeitig als **Lead-Magnet** für That Clicks Marketing:
Nach dem Scan erhält der Nutzer einen Report mit konkreten Problemen und einen
Verweis auf das Beratungsangebot von That Clicks.

---

## Zielgruppe

- Handwerker (Elektriker, Sanitär, Maler, etc.)
- Ärzte, Therapeuten, Heilpraktiker
- Restaurants, Cafés, Gastronomie
- Rechtsanwälte, Steuerberater (kleine Kanzleien)
- Online-Shops (WooCommerce, Shopify, etc.)
- Lokale Dienstleister (Kosmetik, Reinigung, etc.)
- Vereine und Verbände

**Gemeinsamer Nenner:** Keine eigene IT-Abteilung, wenig technisches Wissen,
aber reales DSGVO-Risiko durch veraltete oder falsch konfigurierte Websites.

---

## Wertversprechen

1. **Einfach:** URL eingeben → Ergebnis bekommen. Kein Login, keine Installation.
2. **Verständlich:** Ergebnisse in Klartextsprache ohne Juristenjargon.
3. **Handlungsorientiert:** Jedes gefundene Problem hat eine Empfehlung.
4. **Vertrauenswürdig:** Kein Tracking auf dem Tool selbst, HTTPS, klare Datenschutzerklärung.

---

## Abgrenzung

**Das Tool ist KEIN:**
- Rechtsgutachten oder Rechtsberatung
- Vollständiger DSGVO-Audit
- Ersatz für einen Datenschutzbeauftragten
- Tool für technisch versierte Nutzer (kein Rohdaten-Export im MVP)

---

## Business-Logik

```
Nutzer gibt URL ein
→ Tool scannt technisch (serverseitig)
→ Report mit Ampelfarben (OK / Warnung / Fehler)
→ Am Ende: "Haben Sie Fragen? Wir helfen." → CTA zu that-clicks.de
→ Optionaler Kontakt: info@that-clicks.de
```

---

## Rechtlicher Rahmen

- Das Tool selbst verarbeitet keine personenbezogenen Daten (keine Logins, keine E-Mail-Pflicht)
- Die eingegebene URL wird serverseitig gefetcht und nicht gespeichert
- Klarer Haftungsausschluss in Footer und Report
- Eigene Datenschutzerklärung des Tools vorhanden
