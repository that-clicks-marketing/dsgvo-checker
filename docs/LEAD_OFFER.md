# LEAD_OFFER.md

Texte für den Lead-CTA am Ende des Scan-Reports.
Ton: professionell, hilfreich, nicht aufdringlich. Kein Druck.

---

## Standard-CTA (nach jedem Scan)

### Haupttext

```
Haben Sie Fragen zu Ihren Ergebnissen?

Das DSGVO-Team von That Clicks Marketing hilft Ihnen dabei,
die gefundenen Risiken schnell und unkompliziert zu beheben –
ohne Technik-Chinesisch.
```

### CTA-Button

```
[Kostenlose Erstberatung anfragen]
→ mailto:info@that-clicks.de?subject=DSGVO-Check%20Anfrage&body=Hallo%2C%0D%0Aich%20habe%20Ihren%20DSGVO-Check%20genutzt%20und%20h%C3%A4tte%20gerne%20eine%20kurze%20Beratung.%0D%0A%0D%0AMeine%20Website%3A%20{URL}
```

Oder:

```
[Zur Website von That Clicks Marketing]
→ https://www.that-clicks.de
```

---

## CTA bei vielen Fehlern (6+ Fehler)

```
⚠️ Ihre Website hat mehrere DSGVO-Risiken.

Wir helfen Ihnen, diese gezielt zu beheben – damit Sie auf der
sicheren Seite sind. Vereinbaren Sie jetzt ein kostenloses
Erstgespräch.

[Jetzt anfragen] → info@that-clicks.de
```

---

## CTA bei wenigen Fehlern (0–2 Fehler)

```
✓ Ihre Website sieht gut aus!

Falls Sie sich trotzdem absichern oder weitere Details prüfen
möchten: That Clicks Marketing unterstützt Sie gerne bei einem
vollständigen DSGVO-Audit.

[Mehr erfahren] → https://www.that-clicks.de
```
