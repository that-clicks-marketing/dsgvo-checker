# DSGVO Website Checker

Kostenloser DSGVO-Schnellcheck für kleine Unternehmen – entwickelt von [That Clicks Marketing](https://www.that-clicks.de).

**Live-URL:** https://dsgvo-check.that-clicks.de

---

## Was macht dieses Tool?

Nutzer geben ihre Website-URL ein. Das Tool prüft automatisch auf häufige DSGVO-, Cookie- und Tracking-Risiken und gibt einen verständlichen Report aus – mit konkreten Handlungsempfehlungen.

**Zielgruppe:** Handwerker, Ärzte, lokale Dienstleister, Shopbetreiber, kleine Unternehmen

---

## Tech-Stack

- **Framework:** Next.js 16 (App Router)
- **Sprache:** TypeScript
- **Styling:** Tailwind CSS v4
- **Schriftart:** Montserrat
- **Hosting:** Vercel

---

## Lokal starten

```bash
npm install
npm run dev
```

Dann: http://localhost:3000
